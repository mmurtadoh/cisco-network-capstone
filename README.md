# Cisco Network Capstone

> **End-to-end network design and configuration — subnetting, routing, DHCP, port security, wireless, and Telnet management using Cisco Packet Tracer.**

---

## 📌 Overview

This capstone project demonstrates the design and full configuration of a multi-subnet enterprise network using **Cisco Packet Tracer**. The project covers everything from IP address planning and subnet design, to router/switch configuration, DHCP, wireless access, port security, and remote management via Telnet.

---

## 🛠️ Technologies Used

- Cisco Packet Tracer
- TCP/IP (IPv4)
- VLANs
- DHCP
- Static Routing
- Port Security
- Wireless LAN
- Telnet

---

## 🏗️ What Was Built

### 1. Subnet Design
- Completed a full **subnet worksheet** — calculated network addresses, subnet masks, broadcast addresses, and usable host ranges
- Allocated subnets based on department requirements

### 2. Network Topology
- Designed a complete **network topology** with routers, switches, PCs, and a wireless router
- All links verified green (active/connected)

### 3. Router & Switch Configuration
- Configured routers with IP addressing on all interfaces
- Verified full **routing tables** showing all networks reachable
- Configured inter-VLAN routing where applicable

### 4. DHCP Configuration
- Set up **DHCP pools** with proper address ranges and excluded addresses
- Verified automatic IP assignment on client PCs not directly on the DHCP router

### 5. Port Security
- Implemented **port security** on switches to restrict unauthorized device access
- Tested and verified security violation behavior

### 6. Wireless Access
- Connected a **laptop to a wireless router**
- Verified network connectivity and web page access over wireless

### 7. Remote Management (Telnet)
- Configured and verified a **Telnet session** to a network device
- Demonstrated remote CLI access capability

### 8. Connectivity Testing
- Ran **ping tests** across subnets — verified both successful and expected unsuccessful pings
- Confirmed network segmentation and routing correctness

## 📂 How to Open the Project

1. Download and install **Cisco Packet Tracer** (free via [netacad.com](https://www.netacad.com))
2. Clone or download this repository
3. Open `network-capstone.pkt` in Cisco Packet Tracer
4. Explore the topology, open device CLIs, and run your own ping tests
